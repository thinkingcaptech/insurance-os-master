# Playbook: KPI Architecture (Activity Funnels)

## 1.0 Strategic Mandate

These are the non-negotiable activity funnels for a full-time, "A-Level" agent. An agent's production is a direct, mathematical result of their daily activity. This system is designed to make production **predictable**. All KPIs must be tracked in the CRM dashboard.

---

## 2.0 Final Expense (FE) Funnel

* **Model:** High-volume, high-velocity telesales.
* **Activity Benchmark (Full-Time Agent, Aged Leads):** 150-200 dials/day.
* **Activity Benchmark (Full-Time Agent, Fresh Leads):** 80-120 dials/day.

### Conversion Funnel (Example: 100 Dials on Fresh Leads)

| Stage | Rate (Benchmark) | Funnel (100 Dials) | Source |
| :--- | :--- | :--- | :--- |
| **Dials** | 100% | 100 Dials | --- |
| **Contacts** | 30% - 50% | 40 Conversations | |
| **Appts Set** | 25% - 40% | 12 Appointments Set | |
| **Appts Held** | ~70% | 8 Appointments Held | (Synthesis) |
| **Closes** | 15% - 20% | **1-2 Closes** | |
| **Placement** | 80% - 90% | ~1-2 Issued Policies | |
| **Avg. Premium** | ~$75/mo ($900 AP) | **~$1,350 AP Issued** | |

## 3.0 Mortgage Protection (MP) Funnel

* **Model:** Relationship-driven (B2B) or high-intent digital.
* **Activity Benchmark (B2B):** 5-10 new B2B (Realtor/LO) contacts per day.
* **Activity Benchmark (Digital):** 80-120 dials/day on fresh digital leads.

### Conversion Funnel (Example: 20 B2B Partner Referrals)

| Stage | Rate (Benchmark) | Funnel (20 Referrals) | Source |
| :--- | :--- | :--- | :--- |
| **Referrals** | 100% | 20 Warm Referrals | --- |
| **Contacts** | ~100% | 20 Conversations | (Warm) |
| **Appts Set** | ~90% | 18 Appointments Set | (Warm) |
| **Appts Held** | ~90% | 16 Appointments Held | (Warm) |
| **Closes** | 15% - 25% | **~3-4 Closes** | |
| **Placement** | 70% - 85% | ~3 Issued Policies | |
| **Avg. Premium** | ~$55/mo ($700 AP) | **~$2,100 AP Issued** | |

## 5.0 IUL Funnel

* **Model:** Education-first, long-cycle nurture.
* **Activity Benchmark:** 1 Webinar or 20 LinkedIn DMs per week.

| Stage | Rate (Benchmark) | Funnel (100 Reg.) | Source |
| :--- | :--- | :--- | :--- |
